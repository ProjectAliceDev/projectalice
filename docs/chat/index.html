What is The Studio?

The Studio is our hotspot for communicating between fans, developers, and others alike. We're available through Discord, Matrix.org, and Gitter.im, whatever your preference may be.

Discord

Discord is the original place where all the magic happens in The Studio. Emojis, roles, and more alike, Discord has been a great home for The Studio since inception.

Matrix.org

Matrix is a decentralized messaging system that ties all of The Studio together. Matrix works with any app and doesn't require sign-in to view any of our rooms.

Gitter.im

Gitter.im is a developer-focused chat system that shows live GitHub activity while interacting with The Studio. It's easy to get started by signing in with a GitHub or GitLab account.
